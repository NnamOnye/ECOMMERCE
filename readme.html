Before running this project, you may want to create a super user. Synchronize the Database This project has default settings for the database (it uses sqlite). If you need a super user, you'll need to create one before running the project. Open Solution Explorer. Right-click the project node and select Python->Django Create Superuser. Enter account details when prompted. This creates a new account, and the details do not need to match any of your existing accounts. -------- Django inc.
